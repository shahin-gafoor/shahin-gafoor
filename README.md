## what i'm building right now

**ShiftTracker** — native android app to track work shifts.
started because a colleague was trying to vibe-code a website for the same problem.
figured an android native app would be a great addition —  kotlin, jetpack compose, material 3, foreground service notification, wear os companion in progress.

**DotForge** — self-hosted online ide for c#/.net, built for training sessions at work.
roslyn compiler backend, ~80 concurrent users. because "paste your code into an online compiler" 
wasn't cutting it.

**AI resume tailor** — spring boot + angular app that uses local llm models through ollama to tailor resumes to job descriptions.
generates a pdf too. scratching my own itch while job hunting and also wanted to test out gemma4
